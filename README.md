# Мобильные лабораторные работы

Репозиторий для лабораторных работ по мобильной разработке (Flutter).

## Структура репозитория

```
moblab/
├── lab1_calculator/     # Лабораторная работа 1: Калькулятор
├── lab2_.../            # Лабораторная работа 2
├── lab3_.../            # Лабораторная работа 3
└── README.md
```

## Переключение между лабораторными работами

### Вариант 1: Работа с папками (рекомендуется)

Каждая лабораторная работа находится в отдельной папке. Для работы с конкретной лабораторной работой:

```bash
# Перейти в папку лабораторной работы
cd lab1_calculator

# Запустить проект
flutter run
```

### Вариант 2: Использование веток Git

Если вы хотите использовать ветки Git для каждой лабораторной работы:

```bash
# Создать новую ветку для лабораторной работы
git checkout -b lab2

# Переключиться между ветками
git checkout lab1_calculator  # перейти к лабораторной работе 1
git checkout lab2            # перейти к лабораторной работе 2
git checkout main            # вернуться к основной ветке

# Посмотреть все ветки
git branch
```

## Добавление новой лабораторной работы

1. Создайте новую папку: `labN_название`
2. Скопируйте туда Flutter проект или создайте новый
3. Добавьте изменения в Git:
   ```bash
   git add labN_название/
   git commit -m "Добавлена лабораторная работа N"
   git push
   ```
